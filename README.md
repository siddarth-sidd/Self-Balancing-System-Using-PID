# Self-Balancing-System-Using-PID

Hello Everyone.

This file contains the total overview of making a self balancing system using PID algorithm.
This project mainly discusses the areas of Stabilization of systems using PID.

As this project mainly revolves around stabilization of system, a arduino microcontroller should be enough. I have used arduino uno.
components used are:

<p></p>->arduino uno</p>
<p></p>->L298N motor driver</p>
<p></p>->150rpm BO motors ( 300rpm would be better if it has good torque)</p>
<p></p>->2 Li-ion Batteris</p>
<p></p>->MPU6050 ( This here is the main component)</p>

The structure and alignment of the components are shown in the below image.

<img width="350" height="400" alt="image" src="https://github.com/user-attachments/assets/c02e83c9-1ee4-4e9a-aa63-3799c5ddacbf" />

The Wooden parts are carefully measured and taken as the structure . ( 7cm width, 15 cm height)

<p>As the Structure itself is one of the crucial part to maintain balance. And the MPU6050 , motor driver and microcontroller are aligned at the middle section of the structure and batteries are put at the bottom for the center of mass(it can differ with the structure and wheel placements).</p>
